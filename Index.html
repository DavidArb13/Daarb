FRONT-END

El Frontend se enfoca en el usuario, en todo con lo que podemos interactuar y lo que vemos mientras navegamos. Así como en una primera cita, nuestra web busca causar una buena impresión y agradar al usuario, para lo cual utiliza HTML, CSS y JAVASCRIPT.

BACK-END

El backend es la parte del desarrollo de programación que se encarga de que toda la lógica de una página web funcione, es la parte trasera de cualquier desarrollo web, todo esto gracias a un conjunto de acciones que hacen que todo funione eso es BACK-END.
